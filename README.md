# Fabric recipes for controlling<br />the Go Pro Hero 3 Black camera

Fabric recipes for controlling the Go Pro camera.

These have only been tested with the Go Pro Hero 3 Black

## Requirements

* Python
* [Fabric](http://fabfile.org)
